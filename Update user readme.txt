# Updating User Information
This Sprint Boot MVC application lists the existing list of uses and allows the editing of an existing user. If th provided user id is not valid an error message is display. If the user is edited successfully the list of users is updated and a message is displayed showing the user that his action was successful.
The MySql database is running in a docker container with the help of a docker-compose file and Spring JPA is used as ORM to handle the data access layer.
